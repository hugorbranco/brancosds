    Typography

    Uses Sora for display/headings and DM Sans for body text. All sizes use rem for accessibility.

    Display XL — Aa
    .ds-display-xl 60px / Sora 800
    Display MD — Aa
    .ds-display-md 36px / Sora 700
    Heading XL — Section Title
    .ds-heading-xl 30px / Sora 700
    Heading MD — Card Title
    .ds-heading-md 20px / Sora 600
    Body LG — O Branco's Design System foi criado para facilitar o desenvolvimento de interfaces consistentes.
    .ds-body-lg 18px / DM Sans 400
    Body SM — Texto auxiliar, legendas e descrições secundárias que precisam de menor destaque visual.
    .ds-body-sm 14px / DM Sans 400
    Label — STATUS ATIVO
    .ds-label-lg 12px / DM Sans 600 uppercase
